Class GLine

source code

GObject --+
          |
         GLine

This graphical object subclass represents a line segment. For example, the following code adds lines that mark the diagonals of the graphics window:

       gw = gwindow.GWindow()
       print "This program draws the diagonals on the window."
       gw.add(gobjects.GLine(0, 0, gw.getWidth(), gw.getHeight()))
       gw.add(gobjects.GLine(0, gw.getHeight(), gw.getWidth(), 0))
