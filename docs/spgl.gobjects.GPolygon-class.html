Class GPolygon

source code

GObject --+
          |
         GPolygon

This graphical object subclass represents a polygon bounded by line segments. The GPolygon constructor creates an empty polygon. To complete the figure, you need to add vertices to the polygon using the methods addVertex, addEdge, addPolarEdge. As an example, the following code adds a filled red octagon to the center of the window:

       gw = gwindow.GWindow()
       print "This program draws a red octagon."
       edge = 75
       stopSign = gobjects.GPolygon()
       stopSign.addVertex(-edge / 2, edge / 2 + edge / math.sqrt(2.0))
       for i in range(8):
               stopSign.addPolarEdge(edge, 45i)
       stopSign.setFilled(True)
       stopSign.setColor("RED")
       gw.add(stopSign, gw.getWidth() / 2, gw.getHeight() / 2)
